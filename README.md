# PhDownloader
Download your ph videos

Just clone the repo and add your links to the txt file n run the py script.

Link format - 1.PH links only,
              2.Comma+newline separated eg - 
              
                                             https://ph.com/asda,            
                                             https://ph.com/sdgsdg,                                             
                                             https://ph.com/sdfasfa,
